------------------------------CONTENU du DOSSIER------------------------------------------
Vous trouverez dans cette arborescence tous les fichiers nécessaires à votre mission :
-> la base de données s'appelle PPENR.SQL.

-> sous le dossier assets : 
        les fichiers habituels   + 
        définition d'éléments de présentation supplémentaires : 
          . font supplémentaires
          . toastr
          . des bibliothèques de présentation de tableaux de bord...

-> sous le dossier dossier_conception,
        vous trouverez tous les user case, les scenarii, les maquettes et tous les tests déjà écrits 
        et/ou effectués (n'hésitez à les compléter.)

-> sous le dossier includes
        vous trouverez tous les fichiers REUTILISéS dans toutes les pages pour uniformiser
        la présentation et les traitements

-> sous le dossier back_end
        tous les fichiers d'accès à la base.

-> sous le dossier front_end
        tous les fichiers de descritption des pages en html et php.

-> sous le dossier middlewares :
        des fichiers de vérification si l'utilisateur est bien soit ptrofesseur, soit étudiant. 

 Il est possible qu'il existe des éléments dans assets qui ne sont plus utilisés.
